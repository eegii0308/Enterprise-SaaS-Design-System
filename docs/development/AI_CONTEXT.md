# E-Reconcile MN Project Context

## Product
E-Reconcile MN is a SaaS reconciliation platform for Mongolian businesses.

## Goal
Replace Excel/manual reconciliation processes.

## Current Stage
MVP development.

## Tech Stack
Frontend:
- Next.js
- TypeScript
- Tailwind
- Shadcn UI

Backend:
- Next.js API routes
- PostgreSQL
- Prisma

Authentication:
- NextAuth

Deployment:
- Vercel

## Core Features

1. User management
2. Company workspace
3. Data import
4. Reconciliation engine
5. Matching rules
6. Reports
7. Audit logs
